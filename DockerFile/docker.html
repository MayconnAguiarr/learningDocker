Oque é Docker ?

Docker é a empresa que dirige o movimento de contêiner e o único provedor de plataforma de contêiner para abordar cada aplicativo em toda a nuvem híbrida. As empresas de hoje estão sob pressão para se transformar digitalmente, mas são limitadas pelas aplicações e infraestruturas existentes, enquanto racionalizam um portfólio cada vez mais diversificado de nuvens, datacenters e arquiteturas de aplicativos. O Docker permite uma verdadeira independência entre aplicativos e infra-estrutura e desenvolvedores e operações de TI para desbloquear seu potencial e cria um modelo para melhor colaboração e inovação.

Docker

Os conteiner Docker funcionam em uma única máquina compartilham o kernel do sistema operacional da máquina; eles começam instantaneamente e usam menos computação e RAM. As imagens são construídas a partir de camadas do sistema de arquivos e compartilham arquivos comuns. Isso minimiza o uso do disco e os downloads de imagens são muito mais rápidos.

Padrão

Os conteiner Docker são baseados em padrões abertos e são executados em todas as principais distribuições Linux, Microsoft Windows e em qualquer infraestrutura, incluindo VMs, naked-metal e na nuvem.

Seguro

Os conteiner Docker isolam aplicativos uns dos outros e da infra-estrutura subjacente. O Docker fornece o isolamento padrão mais forte para limitar os problemas do aplicativo a um único recipiente em vez de toda a máquina.
